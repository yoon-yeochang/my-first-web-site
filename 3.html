JavaScript

JavaScript ( English : JavaScript ) is an object -based scripting programming language . This language is mainly used within web browsers and has the ability to access built-in objects of other applications. It is also used for server programming, such as runtime environments such as Node.js. JavaScript was originally developed by Brendan Eich of Netscape Communications Corporation under the name Mocha and later as LiveScript, and eventually became JavaScript. Although JavaScript has similar syntax to Sun Microsystems ' Java , in fact, both languages ​​are based on the basic syntax of C language , and Java and JavaScript have a weak direct relationship
